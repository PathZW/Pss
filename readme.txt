--------------------------------------------------------------------------------
    设备软件平台项目库
--------------------------------------------------------------------------------
    1. 设备软件平台位于 Platform/ 目录.
    2. 包括 SCPI 命令解析与转发、RTOS、lwIP、CMSIS Driver、devDriver 等

--------------------------------------------------------------------------------
    创建新项目
--------------------------------------------------------------------------------
    1. clone Projects 库到“工作目录”, checkout dev, 基于 tags/v1.0.0 创建项目分支
    2. clone Platform 库到“工作目录/Platform/”, checkout dev 分支 (git_clone.sh)
    3. 将整个 Platform/Template 目录复制到 Projects/ 目录, 并改名为"项目名"
    4. 保留需使用的目录并改名为"项目子板名", 然后删除其它目录
    5. 修改镜像发布脚本[image.sh], 确保正确生成镜像文件

--------------------------------------------------------------------------------
    注意事项
--------------------------------------------------------------------------------
    1. [提交上传]或[打包备份]前最好执行 "clear.sh" 清除临时文件.
    2. 不要在项目远端分支上“直接开发”, 而是基于该远端分支创建一个“本地开发分支”.
    3. 推送“本地提交”到“远端分支”时:
       1). 首先执行 git fetch 拉取远端分支更新
       2). 其次执行 git rebase 变基“本地开发分支”提交
       3). 然后执行 git merge 合并“本地开发分支”提交到“远端分支”
       4). 最后执行 git push 推送到“远端分支”
    4. 通常不修改 Platform 文件, 如需修改 Platform 必须保证兼容性, 使用 3 方式提交
