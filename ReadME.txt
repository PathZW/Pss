
--------------------------------------------------------------------------------
注意事项
    1. 编译环境为 KeilMDKv5, 因使用 RTE 组件所以还需安装 KeilMDK 相应组件包
    2. [复制]/[移动]以及[SVN Checkout]等操作后必须执行 "InitRTE.cmd", 也可以在每次 Keil 打开工程前执行一次
    3. [SVN上传]或[打包备份]前最好执行 "clear.cmd" 清除临时文件

--------------------------------------------------------------------------------
目录说明
    ./_MDK/ ---------- KeilMDK 工程与 RTE 相关源代码
    ./_SlickEdit/ ---- SlickEdit 源代码编辑器工程
    ./App/ ----------- 应用相关
    ./Bsp/ ----------- 板级相关
    ./Components/ ---- 开源组件
    ./Middlewares/ --- 中间件
    ./Platform/ ------ 开发平台
    ./clear.cmd ------ 清除临时文件用于 [打包备份] 与 [SVN上传]
    ./InitRTE.cmd ---- 初始化 ./_MDK/RTE/ 目录, 所有修改会自动备份;
    ./Project.axf ---- ARM 编译器生成的可执行文件, 除 BIN 的内容之外，还附加其他调试信息
    ./Project.bin ---- 编译生成的 BIN 可执行文件
    ./Project.hex ---- 编译生成的 HEX 可执行文件
    ./ReadME.txt ----- 本文档
