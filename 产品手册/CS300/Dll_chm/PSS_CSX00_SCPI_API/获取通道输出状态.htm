int PSS_CSX00_ScpiGetOutputStatus(int devID, uchar ch, bool* on,

uint timeout)

函数名：PSS_CSX00_ScpiGetOutputStatus

功  能：获取指定通道输出状态

参  数：devID：输入参数：通信API返回成功的设备ID号

        ch：    输入参数：指定通道号（必须为合法通道号：详见备注）

on：输出参数：设备输出状态，true：已打开，false：已关闭

timeout:输出参数：等待返回结果超时时间(ms)

返回值：函数成功执行返回0，失败返回非0值

备  注：无
