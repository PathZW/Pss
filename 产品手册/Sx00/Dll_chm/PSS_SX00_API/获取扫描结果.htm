int PSS_SX00_GetSweepResult(int devID, int* points, double* vResList, double* iResList, uint timeout)

函数名：PSS_SX00_GetSweepResult

功  能：获取扫描结果

参  数：devID：输入参数：通信API返回成功的设备ID号

        points：输出参数：扫描结果点数（V/I对的个数）

        vResList：输出参数：扫描结果电压数据

        iResList：输出参数：扫描结果电流数据

        timeout：输入参数：等待超时时间(ms)

返回值：函数成功执行返回0，失败返回非0值

备  注：无
