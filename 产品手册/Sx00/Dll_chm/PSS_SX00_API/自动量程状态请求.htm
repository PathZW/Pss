int PSS_SX00_AutoRangeQuery(int devID, bool isVSrc, bool isQuerySrc, bool* on, uint timeout)

函数名：PSS_SX00_ AutoRangeQuery

功  能：自动量程状态请求

参  数：devID：输入参数：通信API返回成功的设备ID号

        isVSrc：输入参数：是否电压源，false表示电流源

        isQuerySrc：输入参数：是否请求源，false表示请求限

        on：输出参数：自动量程状态值

        timeout：输入参数：等待请求值返回超时时间(ms)

返回值：函数成功执行返回0，失败返回非0值

备  注：无
