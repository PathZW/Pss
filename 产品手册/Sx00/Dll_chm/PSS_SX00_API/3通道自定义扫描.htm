int PSS_SX00_Ch3CustomSweep(int masterDevID, int slave1DevID, int slave2DevID, bool isV, unsigned int points, bool limit, double* masterCustomData, double* slave1CustomData, double* slave2CustomData)

函数名：PSS_SX00_Ch3CustomSweep

功  能：开始3通道自定义扫描

参  数：masterDevID：输入参数：通信API返回成功的主设备ID号

        slave1DevID：输入参数：通信API返回成功的从1设备ID号

        slave2DevID：输入参数：通信API返回成功的从2设备ID号

        isV：对否扫描电压；false扫描电流

points：输入参数：待扫描点数

limit：输入参数：是否超限停止；false超限不停止

masterCustomData：输入参数：主设备自定义扫描数据（单位:V/A）

slave1CustomData：输入参数：从1自定义扫描数据（单位：V/A）

slave2CustomData：输入参数：从2自定义扫描数据（单位：V/A）

返回值：函数成功执行返回0，失败返回非0值

备  注：必须打开3台设备,1主2从，且设备之间trig线连接正常
