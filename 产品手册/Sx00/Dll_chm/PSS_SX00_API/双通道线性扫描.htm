int PSS_SX00_Ch2LinearSweep(int masterDevID, int slaveDevID, bool isV, unsigned int points, double startVal, double stopVal, bool limit)

函数名：PSS_SX00_Ch2LinearSweep

功  能：开始双通道线性扫描

参  数：masterDevID：输入参数：通信API返回成功的主设备ID号

        slaveDevID：输入参数：通信API返回成功的从设备ID号

        isV：对否扫描电压；false扫描电流

points：输入参数：待扫描点数

startVal：输入参数：扫描起点值（单位:V/A）

stopVal：输入参数：扫描结束值（单位：V/A）

limit：输入参数：是否超限停止；false超限不停止

返回值：函数成功执行返回0，失败返回非0值

备  注：必须打开2台设备，且设备之间trig线连接正常
