int PSS_SX00_ScpiLimitAutoRangeQuery(int devID, bool isVSrc,

bool* on, uint timeout)

函数名：PSS_SX00_ ScpiLimitAutoRangeQuery

功  能：请求限自动量程状态

参  数：devID：输入参数：通信API返回成功的设备ID号

        isVSrc：输入参数：是否电压源，false表示为电流源

        on：输出参数：自动量程开关，true表示打开；false表示关闭

        timeout：输入参数：等待请求结果超时时间(ms)

返回值：函数成功执行返回0，失败返回非0值

备  注：无
