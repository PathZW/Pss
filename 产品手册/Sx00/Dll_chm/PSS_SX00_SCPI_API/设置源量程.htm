int PSS_SX00_SetSourceRange(int devID, bool isVSrc, double range, bool maxMatch)

函数名：PSS_SX00_SetSourceRange

功  能：设置设备源量程

参  数：devID：输入参数：通信API返回成功的设备ID号

isVSrc：输入参数：设备是否为电压源，false表示设备为电流源

    range：输入参数：待设置的量程值（单位：V/A）

    maxMatch：输入参数：是否最大匹配量程值，false表示向前匹配最佳量程。如量程档位为100mA和1A,

                        用户设置值为200mA,当maxMatch为true时，量程匹配设置为1A，当maxMatch为false时，量程匹配设置为100mA。

返回值：函数成功执行返回0，失败返回非0值

备  注：无
