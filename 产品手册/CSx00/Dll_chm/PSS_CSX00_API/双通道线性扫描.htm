int PSS_CSX00_Ch2LinearSweep(int devID, uint8 masterCh, uint8 slaveCh, bool isMasterV, bool isSlaveV, uint8 masterOutputLine, uint8 slaveSweepLine, unsigned int points, double startVal, double stopVal, bool limit, uint8 trigType)

函数名：PSS_CSX00_Ch2LinearSweep

功  能：开始指定通道双通道线性扫描

参  数：devID：     输入参数：通信API返回成功的设备ID号

        masterCh：   输入参数：主通道号

        slaveCh：    输入参数：从通道号

        isMasterV：输入参数：是否扫描主通道电压；false扫描电流

        isSlaveV：输入参数：是否扫描从通道电压；false扫描电流

        masterOutputLine：输入参数：主设备输出完成触发线

        slaveSweepLine：输入参数：从设备扫描完成触发线

points：输入参数：待扫描点数

startVal：输入参数：扫描起点值（单位:V/A）

stopVal：输入参数：扫描结束值（单位：V/A）

limit：输入参数：是否超限停止；false超限不停止

trigType：输入参数：触发类型：1上升；2下降；3边沿

返回值：函数成功执行返回0，失败返回非0值

备  注：无
