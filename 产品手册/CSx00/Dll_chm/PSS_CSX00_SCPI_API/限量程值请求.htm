int PSS_CSX00_ScpiLimitRangeQuery(int devID, uchar ch, bool isVSrc, char* range, uint timeout)

函数名：PSS_CSX00_ ScpiLimitRangeQuery

功  能：请求指定通道限量程值

参  数：devID：输入参数：通信API返回成功的设备ID号

        ch：    输入参数：指定通道号（必须为合法通道号：详见备注）

        isVSrc：输入参数：是否电压源，false表示为电流源

        range：输出参数：量程值

        timeout：输入参数：等待请求结果超时时间(ms)

返回值：函数成功执行返回0，失败返回非0值

备  注：无
