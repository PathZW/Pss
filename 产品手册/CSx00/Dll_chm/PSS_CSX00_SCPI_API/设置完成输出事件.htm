int PSS_CSX00_ScpiSetFinishOutputEvent(int devID, uint8 ch, uint8 inLine, uint8 outLine, uint8 trigType)

函数名：PSS_CSX00_ ScpiSetFinishOutputEvent

功  能：设置指定通道完成输出事件

参  数：devID：输入参数：通信API返回成功的设备ID号

        ch：输入参数：通道号

        inLine：输入参数：输入事件trig线

        outLine：输入参数：输出事件trig线

        trigType：输入参数：trig触发类型：0：不触发，1：下降沿，2：上升沿，3：边沿

返回值：函数成功执行返回0，失败返回非0值

备  注：trig线只能为0-16,0表示不设置该方向上的事件
